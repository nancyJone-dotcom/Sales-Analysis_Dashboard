# Sales-Analysis_Dashboard

1. Project Title

Sales Analytics & Performance Dashboard

2. Purpose / Description

A Power BI dashboard designed to analyze sales performance, profit trends, customer segments, product categories, and discount impact.
The goal is to provide clear insights that support business decision-making and improve overall sales strategy.
